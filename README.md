# 🚀 DevMaster Pro - Plataforma de Algoritmos

Bem-vindo ao **DevMaster Pro**, uma aplicação web interativa desenvolvida para facilitar o estudo de Lógica de Programação. Esta plataforma organiza o curso de Algoritmos do canal *Curso em Vídeo* em um ambiente focado e produtivo.

## ✨ Funcionalidades

- 🔐 **Sistema de Acesso:** Login personalizado com nome e e-mail.
- 📺 **Playlist Completa:** 15 aulas organizadas de forma sequencial.
- ✍️ **Bloco de Notas Individual:** Cada aula possui seu próprio espaço para anotações, salvas automaticamente no navegador.
- ⏲️ **Timer Pomodoro:** Cronómetro integrado para gerir ciclos de estudo e pausa.
- 🌓 **Temas Dinâmicos:** Suporte para temas Dark (GitHub), Dracula e Matrix.
- 🎯 **Modo Foco:** Interface simplificada para evitar distrações durante o vídeo.
- 🎓 **Certificado Digital:** Geração automática de certificado após a conclusão de 100% das aulas.
- 📥 **Exportação:** Opção para descarregar todas as notas de estudo em formato `.txt`.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura da plataforma.
- **CSS3:** Design responsivo, variáveis CSS para temas e animações.
- **JavaScript (Vanilla):** Lógica de progressão, manipulação do DOM e persistência com `localStorage`.

## 🚀 Como usar

1. Clone este repositório ou baixe o arquivo `index.html`.
2. Abra o arquivo em qualquer navegador moderno.
3. Insira seu nome e e-mail para acessar o painel.
4. Bons estudos!
